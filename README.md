# S2 LITE

컴퓨터에 침입한 악성코드를 퇴치해보자!

## 개발 정보  
개발팀: 소설기 11조<br/>
개발기간: 2021.10 ~ 2021.12 <br/>

> 이 프로젝트는 세종대학교 '소프트웨어설계기초'에서 개발되었습니다.

## 기획

장르 : RPG </br>
규칙 : 컴퓨터에 침입한 악성코드를 퇴치하는 게임<br/>
게임 목표 : 캐릭터 성장 및 보스 퇴치<br/>
